# Public Health Analytics Mini Project

## Overview
This project analyzes US Chronic Disease Indicators (CDI) data to provide insights into public health trends across different states, demographics, and time periods. The analysis leverages data from multiple public health sources to identify patterns in disease incidence, mortality rates, and health behaviors.

## Contents
- **US_CDI_EDA_Final_.ipynb** - Comprehensive exploratory data analysis of the CDI dataset with visualizations and insights
- **US_CDI_Cleaned.csv** - Cleaned dataset ready for analysis (3M+ records)
- **US_Chronic_Desease_Indicator.csv** - Raw CDC public health indicator data
- **Public_Health_Analystics_Dashboard.pbix** - Interactive Power BI dashboard for data visualization
- **Public_Health_Analytics_dashboard.pdf** - Static PDF export of the Power BI dashboard
- **Executive_Summary.pdf** - High-level summary of key findings and insights

## Key Insights
The analysis covers 19 major public health topics including:
- Cancer incidence and mortality
- Diabetes and chronic disease prevalence
- Cardiovascular disease risk factors
- Mental health and disability metrics
- Tobacco, alcohol, and lifestyle factors
- Immunization rates and infectious disease control
- Maternal and child health indicators
- Data spans 2015-2022 across 55 US states/territories

## Technologies Used
- **Python**: pandas, numpy, matplotlib, seaborn
- **Power BI**: Interactive dashboards and data visualization
- **Data Analysis**: Exploratory Data Analysis (EDA), trend analysis, demographic stratification
